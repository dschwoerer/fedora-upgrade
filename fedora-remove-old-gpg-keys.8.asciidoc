NAME
----
fedora-remove-old-gpg-keys - tool to remove old RPM GPG keys


SYNOPSIS
--------
*fedora-remove-old-gpg-keys*


DESCRIPTION
-----------

Script *fedora-remove-old-gpg-keys* removes from rpmdb the GPG keys of EOLed Fedora Linuxes.

--------
rpm(8), fedora-upgrade(8)
